# 🎡 Red Roulette

**Red Roulette** is a polished, homebrew application for the Nintendo Switch. It helps you decide what to play next by randomly picking a game from your installed library. [cite_start]It features smooth physics, custom background/font support, and the ability to build and save custom roulette wheels. [cite: 1, 2, 3]

## 🎮 Controls
| Button | Action |
| :---: | :--- |
| **(A)** | [cite_start]Spin the Roulette! [cite: 1, 2, 3] |
| **(B)** | [cite_start]Re-roll a new default random Roulette [cite: 1, 2, 3] |
| **(-)** | [cite_start]Open the "Saved Roulettes" sidebar [cite: 1, 2, 3] |
| **(R)** | [cite_start]Open the "Create Custom Roulette" grid [cite: 1, 2, 3] |
| **(+)** | [cite_start]Exit the application [cite: 1, 2, 3] |
| **🕹️ Left Stick / D-Pad** | [cite_start]Navigate all menus and selection grids [cite: 1, 2, 3] |

### 🛠️ In Menus:
| Button | Action |
| :---: | :--- |
| **(X)** | [cite_start]**[Sidebar]** Load selected wheel / **[Grid]** Save Custom Wheel [cite: 1, 2, 3] |
| **(Y)** | [cite_start]**[Sidebar]** Delete selected wheel [cite: 1, 2, 3] |
| **(B)** | [cite_start]**[Sidebar/Grid]** Close menu / Cancel [cite: 1, 2, 3] |

## 📥 Installation
[cite_start]Download the **RedRoulette.nro** and place it in the `/switch/` folder on your SD card[cite: 2, 3].

## 📂 Customization
If you want to change background image:
* [cite_start]**Background:** Place a **`bg.jpg`** at `sdmc:/switch/bg.jpg`. [cite: 1, 2, 3]
